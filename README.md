# Lessons in C++

This is a set of lessons to teach `c++` programming. It assumes you know already know how to program. The emphasis is on understanding `c++` data structures, algorithms and how to write fast code in `c++`. The lessons are organized to work with the book [Algorithms](https://www.amazon.com/Algorithms-Sanjoy-Dasgupta/dp/0073523402/ref=sr_1_1?crid=1HNLO2VPENQEP&dib=eyJ2IjoiMSJ9.LO4hCSNXNsy5Gg3qZuUhKFMw--O1YAAmQGgDTHP1ViLCNppEnEdHL0wcT4Pdyb6gp2ZtL14lzC30K0msonvwRsECoXeUgoANXdFgwBp1lEDOv3nf2nx6gW8xITOor5dCdVI2-oxIm8EU1HynlRn2k4rj_o65Hv6ydmMlnCw6KuVsQFJB5PKZzYP9nkLtLw8EqbAMnvDScRRSRVJ4b1BgC8Y_WalHE0kerhjYZWSHRLk.h9qJsZz40Xef-T7AtvwK3aa0j4ZkxZurXCAvtU3RxRk&dib_tag=se&keywords=algorithms+dasgupta&qid=1769195626&sprefix=algorithms+da%2Caps%2C194&sr=8-1) by Dasgupta, Papadimitriou and Vazirani. There are warm up assignments and then additional [**leetcode**](https://leetcode.com/) assignments that should be submitted to **leetcode**. Some assignments have solutions or hints for optimizations.

Lessons are designed to be run on Apple computers.
